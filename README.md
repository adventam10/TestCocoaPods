# TestCocoaPods

自作ライブラリをCocoaPodsで導入したプロジェクトです。

* [SwiftPM](https://github.com/adventam10/TestSwiftPM)
* [Carthage](https://github.com/adventam10/TestCarthage)

## 環境
* Xcode 11.0
* Swift 5
* iOS target 13.0
